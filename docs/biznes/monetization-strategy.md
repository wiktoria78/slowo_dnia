## 4. Pricing Psychology

### 4.1 Strategia Cenowa

| Element | Implementacja |
|---------|---------------|
| **Anchor** | "Normalna cena: 29 PLN" → "Promocja: 19 PLN" |
| **Charm pricing** | 19 PLN (nie 20) |
| **Yearly discount** | 149 PLN/rok = -35% vs monthly |
| **Lifetime** | "Zapłać raz, masz na zawsze: 299 PLN" |

### 4.2 Wording

| Element | Copy |
|---------|------|
| **CTA Premium** | "Odblokuj pełny dostęp" |
| **Social proof** | "Dołączyło 500+ osób" |
| **Risk reversal** | "30 dni gwarancji zwrotu" |
| **FOMO** | "Oferta limitowana: +50 słów" |

## 6. Unit Economics

### 6.1 Kluczowe Metryki

| Metryka | Cel |
|---------|-----|
